## Technologies
* Android Studio
* Kotlin
* Room DB
* Recycler View 
* Firebase Auth
* Firebase Realtime
* Firebase Storage


## Features
* You can create your Meet.
* Share invite code with whatsapp, message etc.
* Share screen feature.
* User can chat with other and raise hand option. 
* It support upto 70 people in a Meeting.
